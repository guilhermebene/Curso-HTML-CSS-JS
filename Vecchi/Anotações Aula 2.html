Tags importantes, seus usos e significados

  • Símbolos, br, parágrafo:

    Símbolo: 𝔸 / Testes (ver comentários): content TODO write content TODO write content TODO write content TODO write content TODO write content TODO write content
    TODO write content TODO write content TODO write content

  • Tag pre: 
                Essa tag representa todos os espaços e quebras de linha usados dentro dela:

            X O O O X
            O X O X O
            O O X O O
            O X O X O
            X O O O X
  • Tag code:
    Apenas demarca que esse trecho representa um trecho de código-fonte em alguma linguagem:
    for int i = 0:10
      disp(i);
    end

    • Ignorar por agora, ver Links e Ancoras

    Brasil



  • Tag i:
    Não significa itálico! Significa coisa que normalmente ficam em itálico:
    • Outro idioma
    • Palavras científicas
    • Muda a entonação em softwares de leitura
    • Etc

  • Tag b:
    Não significa negrito! Usado para palavras-chave

  • Tags strong e em:
    Strong usado para palavras-chave também, que requerem atenção do usuário
    Em tem objetivo é dar ênfase em uma palavra

  • Tags sub e sup:
    Sub subscreve algo CO2
    Sup superscreve algo x2

  • Citações (Diferentes apresentações):
    • blockquote: Citação longa:
      Texto longo
    • q: Citação curta:
      Texto curto

      • Obs: O atributo cite não faz diferença no navegador

    • cite: usado para denotar uma simples citação:
      A mulher sem pecado

  • Abreviações:
    Usado para dar significado para abreviação e siglas
    Ex: IME - Mostra o significado ao passar o mouse

  • Tag dfn: usada quando se dá uma definição
    Ex: Um pingado é um copo de café com leite.

  • Tags ins, del e s: Denotam inserções e remoção de termos. Usado para dar significado e não aparência
    Ex ins e del: Inserido Retirado
    Ex s: Antigo Novo

  • Data e hora:
    Usamos a tag time para demarcar que se fala de uma hora ou data e o atributo datetime se quisermos guardar a data e hora na tag
    Ex: Apenas time mostrando uma data: 2013-12-25 23:59 (aa/mm/dd)
    Demarcando um texto com uma data: Ano Novo 2014

  • Tag mark: Usada para marcação de texto
    Ex: Java e C# são linguagens de programação
    Como esperado, o estilo pode ser editado com CSS
