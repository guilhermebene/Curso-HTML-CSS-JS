Formulários

Vamos criar um formulário com a tag form. Dentro dela teremos diversas opções. Os dados serão obtidos por PHP a partir dos parâmetros action e method.
O parâmetro method pode ser definido como "post" ou "get"

Tags:
  • input type = "text": Caixa de texto. Usamos name (identifica o conteúdo), placeholder (O que aparece escrito para o usuário) e id para definir o significado do input
  • input type = "password": Omite o conteúdo digitado. É exposto na URL caso seja usado o método get. Usar método post caso haja senhas
  • label for = id: Dá o rótulo do input. Importante para acessibilidade, pois ao ser clicado, começa a esperar o input
  • input type = "submit": Botão de envio. Também poderíamos usar a tag button com parâmetro type = "submit"
  • input type = "search": input focado em pesquisa
  • input type = "number": input apenas para números. Possível definir valores mínimo e máximos do intervalo mostrado. Pode-se mudar o step de variação dos números mostrados
  • input type = "range": input com barra selecionável arrastável. Define-se valores para cada posição, mas ele não é mostrado
  • input type = "email": esse input requere formato válido de email
  • input type = "tel": apenas aceita formato de telefone
  • input type = "url": apenas aceita formato de URL, incluindo http://
  • input type = "date": aceita apenas formato de data e abre menu de calendário. A ordem de dd/mm/yy depende do sistema operacional
  • input type = "datetime-local", "month", "week", "time": atuam análogo ao date
  • textarea: campo de texto com tamanho definível e redimensionável pelo usuário. Não é void! Valores entre as tags serão preenchidos na textarea
