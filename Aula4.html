  • Partes de um documento HTML / Agrupamentos: (listados aqui e explicados na apostila, a partir da p. 117)
    • div: define uma divisão genérica do conteúdo. Block-level element. Antes do HTML 5, todos os elementos abaixo eram divs.
      Para maior semântica foram criadas as tags acima
    • span: funciona como div mas é In-Line element
    • header: define um cabeçalho
    • footer: define um rodapé
    • nav: define um menu de navegação
    • article: define um artigo do conteúdo. Deve ser um agrupamento auto-suficiente
    • aside: define um conteúdo secundário do documento
    • section: define uma seção genérica
    • figure: o elemento figure deve ser utilizado para definir um conteúdo que é auto-suficiente e tipicamente referenciado como uma
      unidade singular do fluxo principal do documento. Não deve ser algo cujo article necessite da figure. Normalmente usado para imagens
    • figcaption: coloca legenda na sua tag figure mãe. Deve ser sempre aninhado a uma figure
    • hr: linha horizontal


CSS

Responsável pelo estilo do documento. Podemos usar uma tag style, dentro do html, dentro da tag head para definir para todo o documento.
Podemos usar um parâmetro style para definir o estilo de uma tag. Podemos também usar um arquivo externo, chamada pela tag link na tag head
O estilo pode ser definido para uma certa tag ou pode-se criar um id para um uso específico de uma tag.
Para uma tag, basta usar: nomeDaTag{ }
Caso seja um id, usamos: #nomeDoId{ }
